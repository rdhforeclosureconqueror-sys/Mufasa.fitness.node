F-028
CLOSED
All remaining first-party browser HTML mutation sinks are explicitly inventoried by file, exact occurrence count, and safety rationale. The repository check rejects unreviewed files and any count increase. javascript: form actions and document.write bootstrap loading were removed; script loading now uses createElement with a constant local URL. safeUrl removes ASCII/C1 control characters before scheme detection, rejects javascript:, all general data: URLs, SVG data, and obfuscated scheme forms, and permits only tightly formed base64 PNG/JPEG/GIF/WebP data images when explicitly requested. Rendering tests preserve payloads only as text for script tags, event handlers, SVG, encoded entities, Unicode bidi text, and prototype-pollution key names.
